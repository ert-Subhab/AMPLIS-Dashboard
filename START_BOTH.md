# How to Start Both Applications

## HeyReach Dashboard

**Terminal Command:**
```bash
python app.py
```

**Browser Link:**
http://localhost:5000

**Port:** 5000

---

## Task Manager

**Terminal Command:**
```bash
cd task_manager
python task_manager.py
```

**Browser Link:**
http://localhost:5001/tasks

**Port:** 5001

---

## Running Both at Once

Open **two separate terminal windows**:

**Terminal 1 (HeyReach Dashboard):**
```bash
python app.py
```
Then open: http://localhost:5000

**Terminal 2 (Task Manager):**
```bash
cd task_manager
python task_manager.py
```
Then open: http://localhost:5001/tasks

Both applications can run simultaneously since they use different ports!
