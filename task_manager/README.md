# Task Manager Application

## Quick Start

### Commands to Run

**Windows:**
```bash
cd task_manager
python task_manager.py
```

**Linux/Mac:**
```bash
cd task_manager
python3 task_manager.py
```

Or use the provided scripts:
- Windows: `run_task_manager.bat`
- Linux/Mac: `run_task_manager.sh`

### Browser Links

- **Task Manager**: http://localhost:5001/tasks
- **HeyReach Dashboard**: http://localhost:5000 (run from main directory with `python app.py`)

## Features

- Client-based task organization
- AI-powered task extraction from emails/call logs
- Voice input for task creation
- Notion-like document editor with block-based typing
- Search and filtering
- Date tracking
- Webhook integrations

See TASK_MANAGER_README.md for full documentation.
